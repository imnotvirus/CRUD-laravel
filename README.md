## Um CRUD feito em laravel para aprender as funcionalidades do framework php

-   clone o repositorio
-   composer install
-   configure a .env
-   banco mysql - laravel  - utf8mb4_unicode_ci
-   php artisan migrate
-   php artisan key:generate
-   php artisan db:seed

URL /books  livros
URL /user   escritores
